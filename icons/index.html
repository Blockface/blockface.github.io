Blockface Icon font

How to use:

  • Include on your html head <link rel="stylesheet" href="//static.blockface.co/font/css/Blockface.css">
  • Put icon anywhere you want <i class="icon-blockface"></i>

Note:

  • You may add size to class: jumbo, large, medium, small, xsmall.
    For example: <i class="icon-blockface medium"></i>
  • Basic colors might be added. color-primary, color-secondary, color-white, color-black, color-cloud, color-gray
    For example: <i class="icon-blockface color-primary"></i>
  • Also available for use in your computer. It uses B for the icon.
